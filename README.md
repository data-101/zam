# zam

An Ionic App

### Technologies used

1. Ionic
2. Amplify
3. RapidAPI

### Run

Install Packages

```
npm i
```

aws-export.js needs to be added at /src/aws-expost.js
To run on browser:

```
ionic serve
```

### What features are implemented?

1. Connectivity with Amplify Auth(AWS cognito)
2. REST API using Amplify connected to lambda functios
3. RapidAPI to search for items and display them
